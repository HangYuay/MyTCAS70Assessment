# TCAS70 Student Advisor
### Potisarnpittayakorn School — โรงเรียนโพธิสารพิทยากร

ระบบประเมินและแนะแนวการสมัคร TCAS ปีการศึกษา 2570 สำหรับนักเรียน ม.6 ปีการศึกษา 2569

## Features
- 👤 บันทึกโปรไฟล์ GPAX และ GPA รายวิชา
- 📊 บันทึกคะแนน TGAT / TPAT / A-Level
- 🏆 บันทึกกิจกรรม รางวัล ค่าย จิตอาสา
- 🎯 ประเมินโอกาสรอบ 1 Portfolio และรอบ 3 Admission
- 🏫 เกณฑ์รับสมัครคณะต่างๆ ของมหาวิทยาลัยชั้นนำ
- 📅 ปฏิทิน TCAS70 ครบทุกรอบ
- 🤖 AI Advisor วิเคราะห์โปรไฟล์ส่วนตัว

## Run Locally
```bash
npm install
npm run dev
```
